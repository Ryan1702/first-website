    期中复习¶

    Week 1 Logic¶

    proposition 命题

    • 已经确认T或F

    truth value 真值

    • 求命题是T或F

    truth table 真值表

    negation 取反

    • \(\neg p = not~p\)

    conjunction 和

    • \(p\land q = p~and~q\)

    disjunction 或

    • inclusive or 意为可以都T
    • \(p\lor q = p~or~q\)

    XOR 异或

    • exclusive or 意为不能都是T
    • \(p\oplus q=p~xor~q\)

    一般or表示\(\lor\)而不是\(\oplus\)

    conditional statement 条件语句

    • \(p\rightarrow q\)

      只有当p为T，q为F时，该式子为F

    \(p\rightarrow q\) 的说法

    • 正向 p...q

      p implies q

      p only if q

      p is sufficient for q

    • 逆向 q...p

      q whenever p

      q follows from p

      q if p

      q is necessary for p

    \(p\rightarrow q = \neg p \lor q\)

    converse 逆命题

    • pq交换

    inverse 否命题

    • pq都取反

    contra-positive 逆否命题

    • pq既交换也都取反

    在求这些命题之前可以先把题目转换成if-then格式

    equivalent 等价的

    • 两个命题如果总是有一样的真值，则称他们等价

    一个命题与它的逆否命题等价

    biconditional statement 双条件语句

    • \(p\leftrightarrow q\)

      只有当pq相同时，这个语句才为真

    precedence 优先级

    • \(\neg\)
    • \(\land \lor\)
    • \(\rightarrow \leftrightarrow\)

    tautology 重言式

    • 一个命题恒为T

    contradiction 矛盾式

    • 一个命题恒为F

    logic equivalent 逻辑等价

    • 当 \(p\leftrightarrow q\) 是tautology

    commutative law 交换律

    • \(p\land q = q\land p\)

    associative law 结合律

    • \(p\land (q\land r) = (p\land q)\land r = p\land q\land r\)

    distributive law 分配率

    • \(p\land (q\lor r) = (p\land q)\lor (p\land r)\)

    内外符号相同符合结合律，符号不同符合分配率

    De Morgan's Law

    \(\neg (p\lor q)=\neg p\land \neg q\)

    \(\neg (p\land q)=\neg p\lor \neg q\)

    想要判断是否等价：

    1. 真值表
    2. 逻辑等价语句

    satisfiable

    • 如果有一个取值能使命题为真，则称它是satisfiable

    unsatisfiable = contradiction

    Week 2 Predicates¶

    predicate 谓词

    • proposition function 命题方程

    universal quantification 全称命题

    • for all 可以满足
    • \(\forall\)

    counter-example 反例

    existential quantification 存在命题

    • there exists one 可以满足
    • \(\exists\)

    \(\exists!\) 表示只存在 如果定义域有无穷多个数只能用 \(\forall \exists\) 而不能用 \(\land \lor\)

    优先级

    • \(\forall \exists\)
    • \(\neg\)
    • \(\land \lor\)
    • \(\rightarrow \leftrightarrow\)

    \(\forall x (P(x)\land Q(x)) = (\forall x P(x))\land (\forall x Q(x))\)

    \(\exists x (P(x)\lor Q(x)) = (\exists x P(x))\lor (\exists x Q(x))\)

    De Morgan's laws for quantifiers

    \(\neg (\forall x P(x)) = \exists x \neg P(x)\)

    \(\neg (\exists x P(x)) = \forall x \neg P(x)\)

    The order of quantifiers

    \(\forall x \forall y P(x,y) = \forall y \forall x P(x,y)\)

    \(\exists x \exists y P(x,y) = \exists y \exists x P(x,y)\)

    Week 3 Rule of inference¶

    argument 论证

    • 很多premises前提得出一个conclusion结论
    • 若它是tautology，则论证valid
    • 如果不是，则论证是incorrect reasoning或fallacy

    There are some tautology

    the law of detachment

    1. modus ponens 肯定前键

      \(((p\rightarrow q)\land p)\rightarrow q\)

    2. modus tollens 否定后键

      \(((p\rightarrow q)\land \neg q)\rightarrow \neg p\)

    hypothetical syllogism 假言三段式

    \(((p\rightarrow q)\land (q\rightarrow r))\rightarrow (p\rightarrow r)\)

    disjunctive syllogism 析取三段式

    \(((p\lor q)\land \neg p)\rightarrow q\)

    addition

    \(p\rightarrow (p\lor q)\)

    simplification

    \((p\land q)\rightarrow p\)

    conjunction

    \(((p)\land (q))\rightarrow (p\land q)\)

    universal instantiation

    • 全称命题可以推出存在命题

    universal generalization

    • 全部满足可以推出全称命题

    universal modus ponens

    \(\forall c (\forall x (P(x)\rightarrow Q(x))\land P(c))\rightarrow Q(c)\)

    existential instantiation

    • 存在命题可以推出有部分可以满足

    existential generalization

    • 有一些满足可以推出存在命题

    Week 4 Sets¶

    set 集合

    • unordered + distinct
    • 无序 + 不重复
    • \(a\in A\)

    roster method 列举法

    set builder notations 描述法

    cardinality \(|S|\)

    表示为集合有多少个元素，也就是size

    empty set \(\varnothing\)

    singleton set \(|S|=1\)

    boolean set \(=\{0,1\}\)

    interval 区间

    proper subset 真子集

    • 不包含equal相等情况

    power set

    • 所有子集构成的集合

    \(|P(S)| = 2^{|S|}\)

    cartesian product

    \(A\times B = \{(a,b)|a\in A, b\in B\}\)

    \((a,b)\) is ordered

    \(|A\times B| = |A|\cdot |B|\)

    union 并集

    intersection 交集

    • 如果他们的交集是空集，则他们disjoint

    \(|A\cup B| = |A|+|B|-|A\cap B|\)

    difference 差集

    \(A\setminus B\)

    complement 补集

    \(\overline{A} = U\setminus A\)

    De Morgan's Laws

    \(\overline{A\cap B} = \overline{A}\cup \overline{B}\)

    \(\overline{A\cup B} = \overline{A}\cap \overline{B}\)

    Week 5 Functions¶

    function 方程

    • mappings or transformations
    • 从每一个a映射到恰好一个b
    • \(\forall a\in A, \exists! b\in B,f(a)=b\)

    \(A\) 是domain/source set

    \(B\) 是codomain/target set

    \(b\) 是image of \(a\)

    \(a\) 是preimage of \(b\)

    \((g\circ f)(a) = g(f(a))\)

    以上的前提是f的值域等于g的定义域

    \(G_f=\{(a,b)|a\in A, b=f(a)\}\subseteq A\times B\)

    \(\lfloor x\rfloor\) 下取整

    \(\lceil x\rceil\) 上取整

    \(S\subseteq A\)

    \(f(S)=\{t\in B|\exists s\in S, t=f(s)\}\)

    \(f(S)=\{f(s)|s\in S\}\)

    \(T\subseteq B\)

    \(f^{-1}(T)=\{s\in A|f(s)\in T\}\)

    \(f^{-1}(T)\) 只是一个表示不能说明 \(f\) 有 \(f^{-1}\)

    成为函数的两个要求

    1. \(\forall a\in A, \exists b\in B, f(a)=b\)
    2. \(\forall a_1,a_2\in A, a_1=a_2\rightarrow f(a_1)=f(a_2)\)

    第一个解决from each，第二个解决unique

    injective 单射

    • \(\forall a_1,a_2\in A, f(a_1)=f(a_2)\rightarrow a_1=a_2\)
    • 理解为每一个因变量都只由一个自变量映射得到
    • 与平行与x轴的线的交点数 \(\leq 1\)

    严格单调必定单射

    surjective 满射

    • \(\forall b\in B, \exists a\in A, f(a)=b\)
    • 理解为所有在值域中的数都被映射到
    • 与平行于x轴的线的交点数 \(\geq 1\)

    bijective = injective + surjective 双射

    \(|A| = |B|\leftrightarrow\) 双射

    injective \(|A|\leq |B|\)

    surjective \(|A|\geq |B|\)

    countable

    • cardinality是有限的或者和 \(\mathbb{N}\) 一样

    infinitely countable

    • cardinality和 \(\mathbb{N}\) 一样

    uncountable 不可数的

    有关countable一些性质

    • \(|A|\leq |B|\) 并且 \(B\) 是可数的，则 \(A\) 也可数
    • \(|A|\leq |B|\) 并且 \(A\) 是不可数的，则 \(B\) 也不可数
    • \(A\) 和 \(B\) 都是可数的，\(A\cap B\) 和 \(A\cup B\) 也是可数
    • \(A\) 不可数，\(B\) 可数，则 \(A\setminus B\) 不可数

    • \(A\) 是可数的，\(f\) 是surjective，则 \(B\) 也是可数的

      也就是\(A\) 是可数的，则 \(f(A)\) 也是可数的

    • \(A\) 和 \(B\) 都是可数的，则 \(A\times B\) 是可数的

    \(\mathbb{NZQ}\) 都可数

    \(\mathbb{R}\) 不可数

    Week 6 Number theory¶

    \(a\) devides \(b\) 就是a整除b

    \(a\) 是factor/divisor of \(b\)

    \(b\) 是multiple of \(a\)

    \(a\mid b\)

    \(b\) is divisible by \(a\) 就是b可以被a整除

    \(a\mid b\) 和 \(a\mid c\) 可以推出 \(a\mid (b+c)\)

    \(a\mid b\) 可以推出 \(a\mid bc\) 其中 \(c\) 是整数

    \(d\) 是正整数

    在 \(0\leq r< d\) 时，有 \(a = dq+r\)

    \(q\) 是商quotient

    \(r\) 是余数remainder

    congruent 同余

    如果 \(m\mid (a-b)\)

    \(a\equiv b\pmod{m}\)

    \(a\) is congruent to \(b\) modulo \(m\)

    congruence relation 同余关系

    \(m\) 是modulus

    \(\{mk+a|k\in \mathbb{Z}\}\) 是congruence class of \(a\) modulo \(m\)

    \(a\equiv b\pmod{m}\) 和 \(c\equiv d\pmod{m}\)

    可以推出

    • \(a+c\equiv b+d\pmod{m}\)
    • \(ac\equiv bd\pmod{m}\)
    • \(a^r\equiv b^r\pmod{m}\)

    base \(b\) expansion

    \(m=(d_kd_{k-1}...d_1d_0)_b\)

    \(m=\sum_{j=0}^k d_j b^j\) 其中 \(0\leq d_j< b\)

    fast modular exponentiation 快速幂

    想要求\(a^k\)

    1. 对 \(k\) 进行二进制分解
    2. 算 \(a, a^2, a^4, ...\)
    3. 最后乘在一起

    prime number 质数

    composite number 合数

    1既不是质数也不是合数

    The Fundamental Theorem of Arithmetic

    \(n=p_1^{k_1}p_2^{k_2}...p_m^{k_m}\)

    \(p_i\) 是prime factors

    \(k_i\) 是正整数

    gcd 最大公因数

    如果gcd为1，则他们互质relatively prime/coprime

    The Euclidean Algorithm 求最大公因数

    \(a=r_0, b=r_1\)

    \(a=bq_1+r_2\)

    \(b=r_2q_2+r_3\)

    \(r_2=r_3q_3+r_4\)

    \(...\)

    \(r_{n-2}=r_{n-1}q_{n-1}+r_n\)

    \(r_{n-1}=r_nq_n\)

    最大公因数就是 \(r_n\)

    Bezout's identity 裴蜀定理

    \(a\) 和 \(b\) 是正数

    \(gcd(a,b)=sa+tb\) 存在有一组解\(s,t\)

    Extended Euclidean Algorithm 拓展欧几里得

    求解 \(gcd(a,b)=sa+tb\) 就是直接先算出gcd

    然后再按照欧几里得的算法反向递推回去

    \(r_n=r_{n-2}-r_{n-1}q_{n-1}\)

    然后将 \(r_i\) 全部变为 \(a\) 和 \(b\)

    如果 \(c\) 和 \(m\) 互质，则 \(ac\equiv bc\pmod{m}\rightarrow a\equiv b\pmod{m}\)

    \(ac\equiv bc\pmod{m}\rightarrow a\equiv b\pmod{\frac{m}{gcd(c,m)}}\)

    \(ab\equiv 1\pmod{m}\)

    那么说 \(a\) is invertible modulo \(m\)

    a在模m下可逆元

    \(b\) is an inverse of \(a\) modulo \(m\)

    b是a模m下的逆元

    \(b\equiv a^{-1}\pmod{m}\)

    \(a\) 模 \(m\) 可逆 \(\leftrightarrow\) \(gcd(a,m)=1\)

    求 \(a\) 对 \(b\) 的逆元

    1. 求解 \(1=sa+tb\)
    2. 两边同时取模 \(b\)
    3. \(s\) 就是答案

    pairwise relatively prime/pairwisely coprime 两两互质

    Chinese Remainder Theorem 中国剩余定理

    \(m_i\) 两两互质

    由若干个 \(x\equiv a_i\pmod{m_i}\) 组成的方程组

    \(x=(\sum a_i(M/m_i)y_i)~mod~M\)

    \(y_i\) 是 \(M/m_i\) 对 \(m_i\) 的逆元
