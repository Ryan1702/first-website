# 期中复习

## Week 1 Logic

proposition 命题

- 已经确认T或F

truth value 真值

- 求命题是T或F

truth table 真值表

negation 取反

- $\neg p = not~p$

conjunction 和

- $p\land q = p~and~q$

disjunction 或

- inclusive or 意为可以都T
- $p\lor q = p~or~q$

XOR 异或

- exclusive or 意为不能都是T
- $p\oplus q=p~xor~q$

> 一般or表示$\lor$而不是$\oplus$

conditional statement 条件语句

- $p\rightarrow q$
    
    只有当p为T，q为F时，该式子为F

$p\rightarrow q$ 的说法

- 正向 p...q
    
    p implies q

    p only if q

    p is sufficient for q

- 逆向 q...p
    
    q whenever p

    q follows from p

    q if p

    q is necessary for p

$p\rightarrow q = \neg p \lor q$

converse 逆命题

- pq交换

inverse 否命题

- pq都取反

contra-positive 逆否命题

- pq既交换也都取反

> 在求这些命题之前可以先把题目转换成if-then格式

equivalent 等价的

- 两个命题如果总是有一样的真值，则称他们等价

一个命题与它的逆否命题等价

biconditional statement 双条件语句

- $p\leftrightarrow q$
    
    只有当pq相同时，这个语句才为真

precedence 优先级

- $\neg$
- $\land \lor$
- $\rightarrow \leftrightarrow$

tautology 重言式

- 一个命题恒为T

contradiction 矛盾式

- 一个命题恒为F

logic equivalent 逻辑等价

- 当 $p\leftrightarrow q$ 是tautology

commutative law 交换律

- $p\land q = q\land p$

associative law 结合律

- $p\land (q\land r) = (p\land q)\land r = p\land q\land r$

distributive law 分配率

- $p\land (q\lor r) = (p\land q)\lor (p\land r)$

> 内外符号相同符合结合律，符号不同符合分配率

De Morgan's Law

$\neg (p\lor q)=\neg p\land \neg q$

$\neg (p\land q)=\neg p\lor \neg q$

想要判断是否等价：

1. 真值表
2. 逻辑等价语句

satisfiable

- 如果有一个取值能使命题为真，则称它是satisfiable

> unsatisfiable = contradiction

## Week 2 Predicates

predicate 谓词

- proposition function 命题方程

universal quantification 全称命题

- for all 可以满足
- $\forall$

counter-example 反例

existencial quantification 存在命题

- there exists one 可以满足
- $\exists$

> $\exists!$ 表示只存在
> 如果定义域有无穷多个数只能用 $\forall \exists$
> 而不能用 $\land \lor$

优先级

- $\forall \exists$
- $\neg$
- $\land \lor$
- $\rightarrow \leftrightarrow$

$\forall x (P(x)\land Q(x)) = (\forall x P(x))\land (\forall x Q(x))$

$\exists x (P(x)\lor Q(x)) = (\exists x P(x))\lor (\exists x Q(x))$

De Morgan's laws for quantifiers

$\neg (\forall x P(x)) = \exists x \neg P(x)$

$\neg (\exists x P(x)) = \forall x \neg P(x)$

The order of quantifiers

$\forall x \forall y P(x,y) = \forall y \forall x P(x,y)$

$\exists x \exists y P(x,y) = \exists y \exists x P(x,y)$

## Week 3 Rule of inference

argument 论证

- 很多premises前提得出一个conclusion结论
- 若它是tautology，则论证valid
- 如果不是，则论证是incorrect reasoning或fallacy

There are some tautology

the law of detachment

1. modus ponens 肯定前键

    $((p\rightarrow q)\land p)\rightarrow q$

2. modus tollens 否定后键

    $((p\rightarrow q)\land \neg q)\rightarrow \neg p$

hypothetical syllogism 假言三段式

$((p\rightarrow q)\land (q\rightarrow r))\rightarrow (p\rightarrow r)$

disjunctive syllogism 析取三段式

$((p\lor q)\land \neg p)\rightarrow q$

addition

$p\rightarrow (p\lor q)$

simplification

$(p\land q)\rightarrow p$

conjunction

$((p)\land (q))\rightarrow (p\land q)$

universal instantiation

- 全称命题可以推出存在命题

universal generalization

- 全部满足可以推出全称命题

universal modus ponens

$\forall c (\forall x (P(x)\rightarrow Q(x))\land P(c))\rightarrow Q(c)$

existential instantiation

- 存在命题可以推出有部分可以满足

existential generalization

- 有一些满足可以推出存在命题

## Week 4 Sets

set 集合

- unordered + distinct
- 无序 + 不重复
- $a\in A$

roster method 列举法

set builder notations 描述法

cardinality $|S|$

表示为集合有多少个元素，也就是size

empty set $\varnothing$

singleton set $|S|=1$

boolean set $=\{0,1\}$

interval 区间

proper subset 真子集

- 不包含equal相等情况

power set

- 所有子集构成的集合

$|P(S)| = 2^{|S|}$

cartesian product

$A\times B = \{(a,b)|a\in A, b\in B\}$

> $(a,b)$ is ordered

$|A\times B| = |A|\cdot |B|$

union 并集

intersection 交集

- 如果他们的交集是空集，则他们disjoint

$|A\cup B| = |A|+|B|-|A\cap B|$

difference 差集

$A\setminus B$

complement 补集

$\overline{A} = U\setminus A$

De Morgan's Laws

$\overline{A\cap B} = \overline{A}\cup \overline{B}$

$\overline{A\cup B} = \overline{A}\cap \overline{B}$
